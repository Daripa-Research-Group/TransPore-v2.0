A FEM-FDM Solver for Two-Phase, Multicomponent Transport

Welcome to TransPore v2.0, a robust MATLAB solver for two-phase, multicomponent transport in porous media. [README.md] This version includes advanced modeling for non-Newtonian (shear-thinning) effects of the polymer component. [154, README.md]

Get Started View Gallery

Key Features

Hybrid Solver

Implements a new hybrid FEM-FDM method for modeling and simulation of surfactant-polymer flooding. [162]

Non-Newtonian Model

Includes a dynamic viscosity model to accurately simulate shear-thinning polymer flooding. [161, README.md]

Flexible Configuration

Easily configure grid size, permeability profiles, injection rates, and fluid properties from the master script. [197, 198]

Multiple Geometries

Supports both rectilinear propagation and quarter-five spot well geometries right out of the box. [214-220]
